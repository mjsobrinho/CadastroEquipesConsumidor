# CadastroEquipesConsumidor

## Importante!

Caso não encontre as API's é necessário verificar e configurar  os endereços nas APIs nas controllers, sempre olhando as respectivas portas

- **EquipesController**
- **PessoasController**
- **EquipesPessoasController**

### Configuração das Strings
Certifique-se de definir as seguintes strings em seu código:

### csharp - atual configuração (Porta 7088) pra execução local
string apiEquipes = "https://localhost:7088/api/Equipes";
string apiPessoas = "https://localhost:7088/api/Pessoas";
string apiEquipesPessoas = "https://localhost:7088/api/EquipesPessoas";
